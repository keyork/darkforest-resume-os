## 当前实现的几个关键结论

- 项目以单用户默认档案 `profile_default` 为中心组织所有数据
- AI 能力不是通过环境变量注入，而是通过前端设置页输入后随请求头发送
- JD 匹配、简历生成、JD 解析、简历导入解析均接入了全局任务系统
- 数据库存储的是结构化 JSON 字符串，业务层负责序列化与反序列化
- 当前技术文档描述的是“代码已实现的行为”，不是最初产品设想稿

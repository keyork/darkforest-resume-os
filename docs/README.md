## 当前实现的几个关键结论

- 项目以单用户默认档案 `profile_default` 为中心组织所有数据
- AI 能力通过前端设置页输入，随请求头发送给服务端（不依赖 `.env`）
- JD 匹配、简历生成、JD 解析、简历导入均接入全局任务系统
- 数据库存储结构化 JSON 字符串，业务层负责序列化与反序列化
- 技术文档描述的是”代码已实现的行为”；规划文档描述的是”尚未实现的设计草案”
